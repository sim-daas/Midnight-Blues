## Installation Commands

Run these commands to install the Midnight Wallet SDK:

```bash
cd /home/ubuntu/githubrepos/midnight-blues/backend

npm install @midnight-ntwrk/ledger-v7@7.0.0 \
            @midnight-ntwrk/wallet-sdk-facade@1.0.0 \
            @midnight-ntwrk/wallet-sdk-shielded \
            @midnight-ntwrk/wallet-sdk-unshielded-wallet \
            @midnight-ntwrk/wallet-sdk-hd \
            @midnight-ntwrk/wallet-sdk-dust-wallet \
            @midnight-ntwrk/wallet-sdk-address-format \
            bip39@3.1.0 \
            rxjs@^7.8.1 \
            fp-ts@^2.16.1 \
            io-ts@^2.2.20
```

This will install the packages needed to submit real blockchain transactions.
